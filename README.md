## 🛠 Getting Started

### Prerequisites

- Java Development Kit (JDK) installed (Java 8 or newer recommended)  
- An IDE or code editor that supports Java (e.g. IntelliJ IDEA, Eclipse, VS Code)  
- (Optional) Command-line / terminal access for compilation & execution  

### Installation / Run Instructions

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Rishika-108/Java-DSA.git
   cd Java-DSA


2. **Open in an IDE:**
   Import the folder as a Java project.

3. **Compile & Run a Java file:**
   For example:

   ```bash
   javac Arrays/Example.java
   java Arrays.Example
   ```

4. **Add your own problems or solutions:**

   * Create a new `.java` file under the appropriate folder.
   * Or create a new folder for a topic you’d like to include (e.g. `DynamicProgramming/`, `Recursion/`, etc.).
   * Follow naming conventions and keep consistency with the existing files.
